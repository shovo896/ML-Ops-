### 1️⃣ Terminal open

```
Ctrl + Alt + T
```

---

### 2️⃣ Python আছে কিনা check

```
python3 --version
```

না থাকলে install

```
sudo apt update
sudo apt install python3
```

---

### 3️⃣ Python file create

```
vim test.py
```

---

### 4️⃣ Insert mode এ যাওয়া

```
i
```

---

### 5️⃣ Python code লেখা

```
print("hello world")
```

---

### 6️⃣ Insert mode থেকে বের হও

Keyboard এ

```
Esc
```

---

### 7️⃣ File save + exit

```
:wq
```

তারপর

```
Enter
```

---

### 8️⃣ Python file run

```
python3 test.py
```

---

# Complete command flow (short)

```
vim test.py
i
print("hello world")
Esc
:wq
python3 test.py
```

---

# Output example

```
hello world
```
